# 朋優学院高等学校鉄道研究部 ホームページ

## 利用予定の技術スタック一覧
今回使用したい技術スタックをいちおうまとめておきました。

| 区分 | 採用技術 | 役割・選定理由 |
| :--- | :--- | :--- |
| **Framework** | **Astro** | ベースとなるメタフレームワーク。SSG（静的サイト生成）による爆速な表示と、Markdownの強力なサポートが特徴でお知らせを簡単に追加できるようにする|
| **UI Library** | **Vue.js** | 現役エンジニアチームのメインスキルを活かしたUIコンポーネント開発用。Astro Islandsという機能で部分的に動作できると思う。多分。 |
| **Package Manager**| **pnpm** | 高速・省容量・堅牢なパッケージ管理ツール。Corepack経由で使用し、チーム内の環境依存を防止可能。 |
| **Headless CMS** | **Pages CMS** | 予算0円で導入できるGitベースの管理画面。Web知識のない部員でもブラウザからお知らせ（Markdown）を更新可能。 |
| **Hosting** | **GitHub Pages**| 完全無料の静的サイト配信環境。管理する費用をタダにできる |
| **CI / CD** | **GitHub Actions**| コードのPushやCMSの更新を検知し、自動でビルドからGitHub Pagesへのデプロイまでを完結できる。 |
| **Domain** | **hoyutekken.github.io** | GitHub Pages標準の無料ドメイン。 |
